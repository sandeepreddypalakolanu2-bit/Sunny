# Ishwarya Portfolio

Responsive portfolio website for Ishwarya with animated sections, contact actions, and a small Node backend for contact and newsletter forms.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/sandeepreddypalakolanu2-bit/Sunny)

## Local Development

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Production Build

```bash
npm run build
npm run start
```

## Free Deployment

This project is prepared for Render free hosting with:

- `render.yaml`
- a Node server that binds correctly for deployment
- a health endpoint at `/api/health`

After pushing this repo to GitHub, connect it to Render and deploy the service. Render will assign a free `onrender.com` domain.
